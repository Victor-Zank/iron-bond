# Iron Bond — Design System

## App Info
- **App Name**: Iron Bond
- **Type**: Responsive Web App (Desktop / Tablet / Mobile)
- **Purpose**: IDF preparation platform for Jewish youth in North America

---

## Colors

| Role         | Hex       | Usage                                      |
|--------------|-----------|--------------------------------------------|
| Primary      | `#1A3A2A` | Buttons, navbar, sidebar active, headings  |
| Primary Light| `#2D5C3E` | Hover states, gradients                    |
| Accent       | `#C8A84B` | CTA buttons, badges, highlights            |
| Accent Light | `#E8C96A` | Hover on accent                            |
| Background   | `#F4F2EE` | Page background                            |
| White        | `#FFFFFF` | Cards, sidebar, inputs                     |
| Text         | `#1C1C1C` | Body text                                  |
| Muted        | `#6B7280` | Subtext, labels                            |
| Light        | `#9CA3AF` | Placeholders, disabled                     |
| Border       | `#E0DDD8` | Card borders, dividers                     |
| Error        | `#C0392B` | Error states                               |
| Success      | `#27AE60` | Success states, badges                     |

---

## Typography

| Role           | Font         | Size / Weight          |
|----------------|--------------|------------------------|
| Display/Heading| Bebas Neue   | 32–80px / Regular      |
| Body           | DM Sans      | 15px / 400             |
| Label          | DM Sans      | 11px / 600 uppercase   |
| Caption        | DM Sans      | 12–13px / 400          |

---

## Spacing

| Unit        | Value |
|-------------|-------|
| Base Unit   | 8px   |
| Border Radius (sm) | 8px |
| Border Radius (lg) | 14px |
| Topbar Height | 60px |
| Sidebar Width | 240px |

---

## Components

| Component   | Style |
|-------------|-------|
| Buttons     | Background `#1A3A2A`, white text, 8px radius, 10px 22px padding, 600 weight. Accent variant: `#C8A84B` bg, `#1A3A2A` text. Outline: transparent bg, `#1A3A2A` border |
| Cards       | White bg, `#E0DDD8` border, 14px radius, 20px padding, `0 2px 16px rgba(0,0,0,0.08)` shadow |
| Inputs      | 1.5px `#E0DDD8` border, 8px radius, 11px 14px padding, focus border `#1A3A2A` |
| Navbar      | Fixed top, `#1A3A2A` bg, 60px height, white text/logo |
| Sidebar     | Fixed left, white bg, 240px wide, `#E0DDD8` right border |
